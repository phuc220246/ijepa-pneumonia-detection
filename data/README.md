# Data

Thư mục này chứa metadata CSV — KHÔNG chứa ảnh gốc.

## Tải dữ liệu
- NIH ChestX-ray14: https://www.kaggle.com/datasets/nih-chest-xrays/data
- RSNA Pneumonia: https://www.kaggle.com/c/rsna-pneumonia-detection-challenge

## Các file CSV
- nih_pretrain_50k.csv — 50k ảnh NIH dùng để pretrain
- rsna_train.csv — 18,678 ảnh RSNA train
- rsna_val.csv — 4,003 ảnh RSNA validation
- rsna_test.csv — 4,003 ảnh RSNA test (split cố định)
